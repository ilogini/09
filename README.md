# 🐾 멍이랑 (withbowwow) - 반려동물 산책 경쟁 앱

> "나이키 런 클럽" 스타일의 반려동물 산책 트래킹 & 소셜 경쟁 플랫폼

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 멍이랑 (withbowwow) |
| 유형 | 네이티브 모바일 앱 (iOS / Android) |
| 기술스택 | React Native (Expo) |
| 지도 API | 네이버 지도 API / 카카오맵 API |
| 수익모델 | 광고(카카오 AdFit) + 프리미엄 구독 |
| 최종수정 | 2026-02-12 |

## 폴더 구조

```
기획산출물/
├── proposal.md                    # 마스터 기획서 (사이트맵→reference 참조)
├── README.md                      # 이 파일
│
├── reference/                     # 페이지별 상세 기획 (콘텐츠 배치 이유 포함)
│   ├── 메인페이지.md              # 홈 탭 + 앱 셸 구조
│   ├── 탭1_홈.md                  # 홈 탭 콘텐츠 상세
│   ├── 탭2_소셜.md                # 소셜 탭
│   ├── 탭3_산책.md                # 산책 (GPS 트래킹)
│   ├── 탭4_랭킹.md                # 랭킹
│   ├── 탭5_마이페이지.md          # 마이페이지 (통합 뱃지 시스템)
│   ├── 추가_뱃지컬렉션.md         # 뱃지 전체보기 (챌린지 대체)

│   ├── 추가_산책상세.md           # 산책 기록 상세
│   ├── 추가_알림.md               # 알림 센터
│   ├── 추가_프리미엄.md           # 프리미엄 가입
│   └── 추가_온보딩.md             # 온보딩/로그인
│
├── prototype/                     # HTML 목업 (페이지 간 탭 내비게이션 연결)

│   ├── 소셜.html
│   ├── 랭킹.html
│   ├── 마이페이지.html            # 통합 뱃지 시스템 포함

│   ├── 지도.html
│   ├── 산책상세.html
│   ├── 알림.html
│   ├── 프리미엄.html
│   └── 온보딩.html
│
├── frontend/                      # 프론트엔드 구현 관련 (예정)
├── design/                        # 디자인 시스템 관련 (예정)
├── backend/                       # 백엔드 설계 관련 (예정)
│
├── 멍이랑_컨셉1_시안.html         # 디자인 컨셉 시안
├── 멍이랑_컨셉2_시안.html
```

## 워크플로우

```
proposal.md (마스터 기획서)
    ↓ 사이트맵 참조
reference/ (각 페이지 상세 기획)
    ↓ 콘텐츠 + UX 설계
frontend/ + design/ + backend/ (구현 명세)
    ↓ 취합
prototype/ (HTML 목업으로 검증)
```

## 빠른 시작

```bash
# 메인 목업 열기 (탭으로 전체 페이지 이동 가능)
open 기획산출물/prototype/멍이랑_메인페이지_시안.html
```
