# 멍이랑 (withBowwow) 디자인 컨셉 & 시스템 가이드

> 반려동물과 함께하는 산책을 게이미피케이션으로 즐겁게!
> 컨셉3 기반: **시원하고 청량한 맑은날의 산책**

---

## 1. 디자인 컨셉

### 1-1. 컨셉 키워드

| 키워드 | 영문 | 의미 |
|--------|------|------|
| **청량감** | Refreshing | 맑은 날 산책의 상쾌한 공기감 |
| **생동감** | Vivid | 반려동물과 함께하는 활기찬 에너지 |
| **투명감** | Transparent | 유리처럼 맑고 가벼운 화면 느낌 |
| **개방감** | Airy | 넓은 하늘 아래 자유로운 산책 |

### 1-2. 컨셉 스토리

> 연두빛 라임과 하늘빛 아주르가 만나, 투명한 유리 위에 햇살이 번지는 느낌.
> 가볍고 맑은 공기 속에서 반려동물과 함께 걷는 상쾌한 순간을 담았습니다.
>
> - 라임(Lime Punch)은 풀밭 위의 햇살, 활기찬 산책의 시작을 상징합니다.
> - 아주르(Azure)는 맑은 하늘, 시원한 바람, 신뢰할 수 있는 기록을 의미합니다.
> - 글래스모피즘(반투명 유리 효과)으로 투명하고 가벼운 공기감을 표현합니다.

### 1-3. 앱 성격과 디자인 방향

| 앱의 성격 | 디자인 방향 |
|-----------|------------|
| GPS 산책 트래킹 | 지도 중심의 깔끔한 UI, 실시간 데이터 강조 |
| 게이미피케이션 (NRC 스타일) | 뱃지, 프로그레스바, 랭킹의 시각적 재미 |
| 소셜 경쟁 | 리더보드, 챌린지의 직관적 비교 |
| 반려동물 중심 | 따뜻하고 친근한 감성 + 이모지 활용 |
| 프리미엄 구독 | 고급스러운 그라데이션 배너 |

### 1-4. 타겟 사용자별 디자인 고려

| 타겟 | 연령 | 디자인 포인트 |
|------|------|--------------|
| 열혈 산책러 | 20~30대 | 트렌디한 글래스모피즘, 깔끔한 데이터 표현 |
| 건강 챙기미 | 30~40대 | 큰 숫자로 통계 강조, 읽기 쉬운 텍스트 |
| 경쟁 러버 | 20~30대 | 랭킹 시각화, 뱃지 수집의 시각적 보상 |

## 3. 타이포그래피

### 3-1. 서체

**Pretendard Variable** (메인 서체)
- CDN으로 불러오는 한글 전용 서체
- 가변 폰트로 다양한 굵기 지원
- 깔끔하고 현대적인 느낌

### 3-2. 글씨 크기 체계

| 이름 | 크기 | 굵기 | 사용 예시 |
|------|------|------|----------|
| **Display** | 52px | 가늘게 (300) | 산책 타이머 `00:12:34` |
| **Hero** | 36px | 매우가늘게 (200) | 날씨 온도 `12°` |
| **Heading 1** | 26px | 굵게 (700) | 주간 통계 `12.5km` |
| **Heading 2** | 22px | 굵게 (700) | 결과 제목 `산책 완료!` |
| **Logo** | 19px | 가늘게+굵게 | **멍이**랑 |
| **Title** | 16px | 중굵게 (600) | 섹션 제목 `산책 시작하기` |
| **Subtitle** | 15px | 중굵게 (600) | 카드 헤더 `이번 주 산책` |
| **Body** | 14px | 중굵게 (600) | 이름, 장소명 `초코`, `보라매공원` |
| **Caption** | 12px | 보통 (400) | 라벨 `주간 목표 20km` |
| **Small** | 11px | 보통 (400) | 상세정보 `골든 리트리버 · 3살` |
| **Tiny** | 10px | 중간 (500) | 네비 라벨 `홈` `소셜` `산책` |

### 3-3. 숫자 표현

- 타이머, 거리, 통계 숫자는 **고정폭 숫자** (tabular-nums) 사용
- 숫자가 변해도 레이아웃이 흔들리지 않도록 함

## 6. 글래스모피즘 (핵심 디자인 언어)

앱의 투명하고 시원한 느낌을 만드는 핵심 효과.
배경이 흐릿하게 비치는 반투명 유리 느낌입니다.

### 6-1. Glass 종류

| 종류 | 투명도 | 블러 | 사용처 |
|------|--------|------|--------|
| **Glass (기본)** | 65% 흰색 | 20px | 일반 카드, 펫카드, 챌린지카드 |
| **Glass Heavy (강한)** | 80% 흰색 | 20px | 헤더, 네비, 산책 패널, 맵 버튼 |
| **Header Glass** | 70% 흰색 | 24px | 상단 헤더 전용 |
| **Nav Glass** | 72% 흰색 | 24px | 하단 네비게이션 전용 |

### 6-2. Glass 테두리

연두색 계열의 아주 연한 테두리: `rgba(197, 217, 0, 0.12)`
카드에 은은한 경계를 주면서도 무겁지 않게 표현

## 9. 컴포넌트 가이드

### 9-1. 카드

| 카드 종류 | 둥글기 | 특징 |
|----------|--------|------|
| 날씨 카드 | 24px | 하늘색 그라데이션 배경 + 구름 애니메이션 |
| 주간 요약 카드 | 24px | 통계 3분할 + 프로그레스 바 |
| 펫 카드 | 20px | 가로 스크롤, 이모지 아바타, 선택 시 연두 테두리 |
| 챌린지 카드 | 20px | 가로 스크롤, 아이콘 + 진행률 바 |
| 산책 기록 카드 | 14px | 가로 배치 (아이콘 + 날짜 + 거리/시간/칼로리) |

### 9-2. 버튼

| 버튼 종류 | 모양 | 색상 | 사용처 |
|----------|------|------|--------|
| **산책 시작 (CTA)** | 라운드 직사각형 | 연두 테두리 (호버 시 채움) | 홈 화면 메인 버튼 |
| **GPS 시작** | 라운드 직사각형 | 연두 그라데이션 채움 | 산책 화면 시작 버튼 |
| **저장** | 라운드 직사각형 | 하늘 채움 | 결과 저장 |
| **정지** | 큰 원형 (64px) | 빨강 그라데이션 | 산책 중지 |
| **일시정지/카메라** | 작은 원형 (48px) | 글래스 반투명 | 산책 컨트롤 |
| **닫기** | 라운드 직사각형 | 연한 회색 | 모달 닫기 |
| **네비 중앙** | 원형 (54px) | 연두 그라데이션, 위로 돌출 | 산책 바로가기 |

### 9-3. GPS 상태 표시 (Pill)

| 상태 | 모양 |
|------|------|
| **GPS 대기** | 회색 점 + "GPS 대기" (연한 배경) |
| **GPS 활성** | 연두 깜빡이는 점 + "GPS 활성" (연두 배경) |

### 9-4. 뱃지 / 칩

| 종류 | 배경색 | 글자색 | 사용처 |
|------|--------|--------|--------|
| Azure 뱃지 | 연한 하늘 | 진한 하늘 | 산책 달성 뱃지 (1km, 3km, 30분) |
| Lime 뱃지 | 연한 연두 | 진한 연두 | 키워드 태그 |

### 9-5. 탭 (Tabs)

| 상태 | 모양 |
|------|------|
| **비활성** | 회색 글자, 굵기 500 |
| **활성** | 검정 글자, 굵기 600, 하단 연두색 2px 밑줄 |

### 9-6. 프로그레스 바

| 항목 | 값 |
|------|---|
| 높이 | 4px (기본), 3px (챌린지) |
| 트랙 색상 | 아주 연한 연두 (12%) |
| 바 색상 | 연두 → 하늘 그라데이션 |
| 전환 | 1.2초 부드러운 애니메이션 |

### 9-7. 토스트 알림

| 항목 | 값 |
|------|---|
| 배경 | 진한 남색 반투명 88% |
| 글자 | 흰색 13px |
| 위치 | 화면 하단 92px |
| 모양 | 둥근 알약형 (24px) |
| 지속시간 | 2.5초 후 사라짐 |

### 9-8. 랭킹 리스트

| 항목 | 설명 |

## 15. 디자인 원칙 5가지

### 01. 투명감 (Transparency)
반투명 유리(글래스모피즘)로 레이어 사이에 깊이감을 만듭니다.
배경이 살짝 비치는 느낌으로 가볍고 시원한 화면을 구성합니다.

### 02. 청량감 (Freshness)
라임 + 아주르 조합, 밝은 배경, 가벼운 빛 효과.
맑은 날 산책의 상쾌함을 색상으로 전달합니다.

### 03. 가벼움 (Lightness)
얇은 글씨체(Light/Thin), 최소한의 테두리, 넉넉한 여백.
무겁지 않고 시원한 화면을 만듭니다.

### 04. 생동감 (Vivid)
떠다니기, 맥박, 구름 흘러가기 등 자연스러운 움직임.
반려동물과 함께하는 활기찬 산책의 에너지를 담습니다.

### 05. 일관성 (Consistency)
같은 색, 같은 둥글기, 같은 간격을 반복 사용.
어디를 봐도 '멍이랑'다운 통일된 경험을 제공합니다.
