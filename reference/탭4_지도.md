# 지도 탭 (Tab 4) - 멍이랑 (withbowwow)

> 하단 탭 바 4번째 탭. 기존 랭킹 탭을 대체.
> 주변 반려동물 친화 장소(공원, 카페, 상점, 동물병원, 약국)를 전체 화면 지도로 탐색.

---

## 탭 위치

| 탭 번호 | 아이콘 | 이름 |
|--------|--------|------|
| Tab 1 | 🏠 | 홈 |
| Tab 2 | 👥 | 소셜 |
| Tab 3 | 🐾 | 산책 (중앙 플로팅) |
| **Tab 4** | **📍** | **지도 (현재 탭)** |
| Tab 5 | 👤 | MY |

---

## 진입 경로

| 경로 | 설명 |
|------|------|
| 하단 탭 바 "지도" 탭 | 어느 화면에서든 직접 접근 |
| 홈 → "내 주변" → "지도보기" | 홈의 미니 지도에서 전체 지도로 확장 |

> **참고**: 두 경로 모두 동일한 화면으로 연결됩니다.

---

## 핵심 기능 요약

| 기능 | 설명 |
|------|------|
| 전체 화면 지도 | 네이버 지도 API 기반, 현재 위치 중심 |
| 장소 핀 | 공원 🌳 / 카페 ☕ / 상점 🛒 / 동물병원 🏥 / 약국 💊 |
| 산책 코스 오버레이 | 인기 경로 Polyline + 히트맵 모드 |
| 필터 시스템 | 카테고리별 표시/숨김 토글 |
| 장소 상세 바텀시트 | 반려동물 특화 정보 (대형견 입장, 리드줄 필수 등) |
