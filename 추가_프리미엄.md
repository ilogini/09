# 프리미엄 가입 스크린 - 멍이랑 (withbowwow)

> 무료 사용자를 프리미엄 구독자로 전환하는 핵심 수익화 페이지.
> Apple Fitness+, Strava Summit 구독 화면의 전환율 최적화 패턴을 적용.

---

## 1. 스크린 개요

### 1.1 목적

프리미엄 가입 스크린은 무료 사용자에게 프리미엄 구독의 가치를 시각적, 논리적으로 설득하여 유료 전환(Conversion)을 유도하는 수익화 핵심 페이지이다. 단순한 기능 나열이 아닌, 사용자가 이미 경험한 무료 기능의 한계점을 인지시키고, 프리미엄이 그 한계를 해소해줌을 구체적으로 보여주는 설득 흐름으로 설계한다. 통합 뱃지 시스템에서 프리미엄 전용 뱃지를 새로운 전환 동력으로 활용한다.

### 1.2 핵심 사용자 니즈

| 니즈 | 설명 | 우선순위 |
|------|------|---------|
| 가치 확인 | "프리미엄이 월 3,900원 값어치를 하는지 알고 싶다" | ✅ 필수 |
| 혜택 비교 | "무료와 프리미엄의 차이를 명확히 비교하고 싶다" | ✅ 필수 |
| 가격 확인 | "정확한 가격과 결제 방식을 알고 싶다" | ✅ 필수 |
| 해지 안전 | "언제든 해지할 수 있는지 확인하고 싶다" | ✅ 필수 |

### 1.3 진입 경로

```
경로 1: 홈 > 프리미엄 배너 탭
경로 2: 마이페이지 > "프리미엄으로 업그레이드" 버튼
경로 3: 잠금 기능 시도 시 > "프리미엄 전용" 팝업 > "자세히 보기"
경로 4: 뱃지 컬렉션 > 프리미엄 전용 뱃지 탭 > "프리미엄 가입"
경로 5: 산책 상세 > 프리미엄 통계 잠금 > "잠금 해제"
```

## 2. 화면 구성

### 섹션 1: 히어로 섹션

#### 콘텐츠 구성

- **배경**: Primary → Secondary 그라데이션 (상단→하단)
- **프리미엄 로고**: 앱 로고 + 금색 왕관 아이콘 오버레이
- **헤드라인**: "멍이랑 프리미엄" (28pt Bold, White)
- **서브 헤드라인**: "초코와의 산책을 더 특별하게" (16pt Regular, White 90%)
  - (실제 사용자의 반려동물 이름을 동적으로 삽입)
- **핵심 혜택 요약** (아이콘 + 텍스트 3줄):
  - "광고 없는 깔끔한 산책 기록"
  - "상세 통계와 건강 리포트"
  - "프리미엄 전용 뱃지 컬렉션"
- **CTA 버튼**: "무료로 7일 체험하기" (White 배경, Primary 텍스트, 18pt Bold, 라운드 48px)
- **서브 텍스트**: "7일 무료 체험 후 월 3,900원 / 언제든 해지 가능" (12pt, White 70%)

#### 이 콘텐츠가 있는 이유

> Spotify Premium, YouTube Premium, Strava Summit 등 성공적인 구독 서비스의 가입 페이지는 모두 시각적으로 임팩트 있는 히어로 섹션으로 시작한다.
>
> - **개인화된 메시지**: "초코와의 산책을 더 특별하게"처럼 사용자의 반려동물 이름을 삽입하면, 일반적인 마케팅 메시지보다 개인적 관련성(Personal Relevance)이 높아져 메시지 수용도가 증가한다. HubSpot 연구에 따르면 개인화된 CTA는 비개인화 CTA보다 202% 높은 전환율을 보인다.
> - **무료 체험(Free Trial)의 설득력**: "무료로 7일 체험하기"는 구매 결정의 심리적 장벽을 낮추는 가장 효과적인 전략이다. 사용자는 "일단 써보고 결정하자"라는 낮은 리스크 인식으로 체험을 시작하며, 7일간 프리미엄에 익숙해지면 유료 전환율이 크게 높아진다 (일반적으로 무료 체험 후 60-70% 유지).
> - **해지 가능성 명시**: "언제든 해지 가능"을 히어로 섹션에 미리 명시하면, 사용자의 "과금 부담" 걱정을 초기에 해소하여 가입 결정까지의 심리적 마찰을 줄인다.

### 섹션 2: 무료 vs 프리미엄 비교 표

#### 콘텐츠 구성

- **레이아웃**: 2열 비교 테이블 (좌측: 무료, 우측: 프리미엄)
- **헤더**: "무료" (Gray 배경) vs "프리미엄" (Primary 배경 + 왕관)
- **비교 항목**:

| 기능 | 무료 | 프리미엄 |
|------|------|---------|
| 산책 기록 | 기본 기록 (거리, 시간) | 상세 기록 (속도 컬러맵, 고도, 걸음 수) |
| 통계 분석 | 주간 요약만 | 일/주/월/연간 상세 분석 + 트렌드 |
| 랭킹 | 주간 랭킹만 | 실시간 랭킹 + 지역별 + 전국 |
| 광고 | 광고 노출 | 광고 완전 제거 |
| 뱃지 | 기본 뱃지 (36종) | 기본 + 프리미엄 전용 뱃지 (47종+) |
| 산책 코스 | 추천 코스 보기만 | 커스텀 코스 생성 + 즐겨찾기 |
| 건강 리포트 | X | 월간 건강 리포트 (체중, 활동량 분석) |
| 공유 카드 | 기본 테마 1종 | 프리미엄 테마 3종 + 커스터마이즈 |

- **강조 표시**: 프리미엄 열의 각 항목 옆에 체크마크(Primary) + 무료 열의 제한 항목에 X 또는 자물쇠
- **하이라이트 행**: "프리미엄 전용 뱃지" 행에 배경색 강조 (연한 민트)

#### 이 콘텐츠가 있는 이유

> Apple Music, Netflix, Notion 등 모든 성공적인 프리미엄 서비스는 무료/유료 비교 표를 핵심 전환 도구로 활용한다.
>
> - **앵커링 효과(Anchoring Effect)**: 무료 기능의 제한사항을 먼저 보여준 뒤 프리미엄의 풍부함을 보여주면, 사용자는 무료의 제한을 "앵커"로 인식하여 프리미엄의 가치를 더 높게 평가한다. 이는 Daniel Kahneman의 앵커링 편향 연구에서 검증된 심리적 메커니즘이다.
> - **손실 회피(Loss Aversion)**: 무료 체험을 경험한 사용자에게 비교 표를 보여주면, 프리미엄 기능을 "앞으로 잃게 될 것"으로 인식하게 하여 해지 방지 효과가 있다. 사람들은 이득보다 손실에 약 2배 민감하게 반응한다.
> - **뱃지 통합의 수익 효과**: 기존 "무제한 챌린지" 대신 "프리미엄 전용 뱃지"로 전환함으로써, 수집 충동을 가진 사용자에게 더 강력한 구독 동기를 제공한다. "47개 중 11개가 프리미엄 전용"이라는 정보는 100% 컬렉션을 원하는 사용자에게 결정적인 전환 요인이 된다.

### 섹션 3: 핵심 혜택 상세 카드

#### 콘텐츠 구성

- **레이아웃**: 수직 스크롤 카드 리스트 (카드 간격 16px)
- **각 카드 구성** (4개):

**카드 1: 상세 통계 & 리포트**
- 아이콘: 차트 아이콘 (64px)
- 제목: "나와 초코의 산책을 더 깊이 이해해요" (16pt Bold)
- 설명: "구간별 속도 분석, 월간 활동 트렌드, 반려동물 건강 리포트까지. 데이터로 더 건강한 산책을 만들어요." (14pt, Gray)
- 비주얼: 통계 화면 목업 이미지 (라운드 코너)

**카드 2: 프리미엄 전용 뱃지**
- 아이콘: 왕관 뱃지 아이콘 (64px)
- 제목: "특별한 뱃지를 수집하세요" (16pt Bold)
- 설명: "프리미엄 회원만 도전할 수 있는 11종의 특별 뱃지. 마라토너, 탐험왕, 시즌 마스터 등 희귀 뱃지로 컬렉션을 완성하세요." (14pt, Gray)
- 비주얼: 프리미엄 뱃지 그리드 미리보기 (글로우 효과)

**카드 3: 광고 제거**
- 아이콘: 슬래시가 그어진 광고 아이콘 (64px)
- 제목: "산책에만 집중하세요" (16pt Bold)
- 설명: "홈 화면, 산책 완료 화면의 모든 광고가 사라집니다. 깔끔한 인터페이스로 산책 기록에만 집중할 수 있어요." (14pt, Gray)

**카드 4: 실시간 랭킹 + 커스텀 코스**
- 아이콘: 트로피 + 지도 아이콘 (64px)
- 제목: "경쟁하고, 새로운 코스를 만들어요" (16pt Bold)
- 설명: "실시간으로 업데이트되는 전국 랭킹과 나만의 산책 코스를 만들어 다른 산책러들과 공유하세요." (14pt, Gray)

#### 이 콘텐츠가 있는 이유

> Headspace, Calm 등 웰니스 구독 앱에서 각 프리미엄 혜택을 카드 형태로 시각적으로 보여주는 패턴은 전환율을 높이는 검증된 방식이다.
>
> - **구체적 혜택의 힘**: "프리미엄이 더 좋습니다"라는 추상적 메시지보다, "구간별 속도 분석", "11종 특별 뱃지"라는 구체적 혜택이 설득력이 높다. Robert Cialdini의 연구에 따르면, 구체적 숫자를 포함한 메시지는 추상적 메시지보다 43% 높은 설득력을 보인다.
> - **감정적 카피라이팅**: "나와 초코의 산책을 더 깊이 이해해요"는 기능 설명이 아닌 감정적 가치를 전달한다. 사용자는 기능(통계 분석)이 아닌 결과(반려동물과의 더 깊은 이해)에 비용을 지불한다.
> - **뱃지 카드의 전략적 배치**: "프리미엄 전용 뱃지" 카드를 두 번째에 배치한 것은, 뱃지 컬렉션에서 유입된 사용자(가장 전환 가능성 높은 세그먼트)가 빠르게 핵심 혜택을 확인할 수 있도록 하기 위함이다.

### 섹션 4: 가격 플랜

#### 콘텐츠 구성

- **레이아웃**: 2개 카드 수평 배치 (좌: 월간, 우: 연간)
- **월간 플랜 카드**:
  - 제목: "월간" (16pt Bold)
  - 가격: "월 3,900원" (24pt Bold)
  - 세부: "매월 자동 갱신" (12pt, Gray)
  - 선택 테두리: 기본 Gray 테두리
- **연간 플랜 카드** (추천 표시):
  - 배지: "33% 할인" (빨간 리본, 우상단)
  - 제목: "연간" (16pt Bold)
  - 가격: "월 2,600원" (24pt Bold, Primary)
  - 세부: "연 31,200원 일시불" (12pt, Gray)
  - 절약액: "연 15,600원 절약!" (14pt, Primary Bold)
  - 선택 테두리: Primary 컬러 두꺼운 테두리 (기본 선택)
- **선택 인터랙션**: 카드 탭으로 플랜 선택, 선택된 카드에 체크마크

#### 이 콘텐츠가 있는 이유

> Netflix, Spotify, YouTube Premium 등 모든 구독 서비스가 월간/연간 플랜을 병렬 제시하며, 연간 플랜에 할인을 적용하는 것은 업계 표준 가격 전략이다.
>
> - **데코이 효과(Decoy Effect)**: 월간 플랜은 연간 플랜의 "가치를 높여주는 비교 대상(데코이)" 역할을 한다. 월 3,900원 vs 월 2,600원을 나란히 보면, 연간 플랜이 "같은 서비스를 더 저렴하게"라는 가치 제안이 명확해진다.
> - **기본 선택(Default Bias)**: 연간 플랜을 기본 선택 상태(Pre-selected)로 설정하면, 행동경제학의 기본값 효과에 따라 사용자가 별도 행동 없이 연간 플랜을 선택하는 비율이 높아진다. Strava에서도 연간 플랜을 기본 선택으로 제시한다.
> - **절약액 명시**: "연 15,600원 절약!"이라는 구체적 숫자는 절약의 크기를 즉시 인지하게 한다. 사람들은 추상적 할인율(33%)보다 구체적 금액(15,600원)에 더 강하게 반응한다.

### 섹션 5: 결제 버튼 및 신뢰 요소

#### 콘텐츠 구성

- **CTA 버튼**: "프리미엄 시작하기" (전체 너비, Primary 배경, White 텍스트, 18pt Bold, 높이 56px)
  - 무료 체험 선택 시: "무료 체험 시작 → 7일 후 월 3,900원"
  - 연간 선택 시: "연간 플랜 시작 → 31,200원"
- **결제 수단**: iOS IAP / Google Play Billing (OS 네이티브 결제 시트 호출)
- **신뢰 요소** (CTA 하단에 소형 텍스트):
  - "언제든 해지 가능 | 위약금 없음"
  - "결제는 Apple/Google을 통해 안전하게 처리됩니다"
  - "개인정보는 안전하게 보호됩니다"
- **결제 보안 아이콘**: 자물쇠 아이콘 + "보안 결제" 텍스트

#### 이 콘텐츠가 있는 이유

> 모든 구독 서비스의 결제 버튼 주변에는 "신뢰 요소(Trust Signals)"가 배치되어 있다. Amazon의 "안전한 결제" 뱃지, Apple의 "Privacy" 아이콘 등이 대표적이다.
>
> - **전환 마지막 장벽 제거**: 사용자가 히어로, 비교 표, 혜택 카드를 모두 보고 "가입하고 싶다"는 마음이 들었어도, 결제 버튼을 누르는 순간에 "과금 불안"이 발생할 수 있다. "언제든 해지 가능", "위약금 없음"은 이 마지막 장벽을 제거하는 역할을 한다.
> - **CTA 문구의 구체성**: "프리미엄 시작하기" 대신 "무료 체험 시작 → 7일 후 월 3,900원"처럼 구체적인 내용을 버튼에 명시하면, 사용자가 "이 버튼을 누르면 무엇이 일어나는지"를 정확히 알 수 있어 불안감이 감소한다.
> - **iOS/Android 네이티브 결제**: 앱 내 결제를 OS 네이티브 시트(IAP / Google Play Billing)로 처리하면, 사용자가 이미 익숙한 결제 UI를 사용하게 되어 신뢰도가 높아지고, 앱 스토어 정책도 준수할 수 있다.

| 질문 | 답변 요약 |
|------|----------|
| 무료 체험 중 해지하면 비용이 청구되나요? | 아니요. 7일 이내 해지 시 비용이 발생하지 않습니다. |
| 결제는 어떻게 하나요? | Apple App Store 또는 Google Play Store를 통해 결제됩니다. |
| 해지는 어디서 하나요? | iOS: 설정 > Apple ID > 구독, Android: Play 스토어 > 구독에서 해지할 수 있습니다. |
| 가족/다른 기기에서도 사용 가능한가요? | 동일 계정으로 로그인하면 모든 기기에서 사용 가능합니다. |
| 프리미엄 전용 뱃지는 해지 후에도 유지되나요? | 네, 이미 획득한 뱃지는 해지 후에도 영구 보존됩니다. 단, 새로운 프리미엄 뱃지 획득은 불가합니다. |
| 월간에서 연간으로 변경할 수 있나요? | 네, 마이페이지 > 설정 > 구독 관리에서 변경할 수 있습니다. |

- **아코디언 동작**: 질문 탭 시 답변 영역 슬라이드 다운, 다른 질문 탭 시 기존 답변 접힘

#### 이 콘텐츠가 있는 이유

> Apple, Amazon, Netflix 등의 구독 결제 페이지에는 FAQ 섹션이 필수적으로 포함되어 있다.
>
> - **불확실성 감소(Uncertainty Reduction)**: FAQ는 사용자가 결제 전에 가질 수 있는 모든 의문점을 사전에 해소한다. Shannon과 Weaver의 커뮤니케이션 이론에 따르면, 불확실성이 낮을수록 의사결정 속도가 빨라진다.
> - **해지 관련 질문의 전략적 배치**: "무료 체험 중 해지하면?"을 첫 번째 질문으로 배치한 것은, 이것이 사용자의 가장 큰 우려 사항이기 때문이다. 이 불안을 초반에 해소하면 나머지 FAQ를 읽으면서 가입 의향이 강화된다.
> - **뱃지 영구 보존 정책**: "해지 후에도 뱃지 유지"라는 정보는 사용자에게 "최소 1개월만 구독해도 뱃지는 영구히 내 것"이라는 인식을 주어, 가입 결정의 리스크를 더욱 낮춘다.

---

### 섹션 7: 해지 안내

#### 콘텐츠 구성

- **해지 경로 안내**:
  - iOS: 설정 > [사용자 이름] > 구독 > 멍이랑 프리미엄 > 구독 취소
  - Android: Google Play 스토어 > 메뉴 > 구독 > 멍이랑 프리미엄 > 구독 취소
- **해지 후 변화 안내**:
  - 현재 결제 기간 종료까지 프리미엄 기능 유지
  - 기간 종료 후 무료 플랜으로 자동 전환
  - 산책 기록, 획득 뱃지, 프로필 데이터 모두 유지
  - 프리미엄 전용 통계/기능만 잠금 처리
- **스타일**: 소형 텍스트 (12pt, Light Gray), 페이지 최하단 배치

#### 이 콘텐츠가 있는 이유

> App Store 리뷰 가이드라인과 Google Play 정책에 따라, 구독 해지 방법을 앱 내에서 명확히 안내해야 한다.
>
> - **규정 준수**: Apple App Store Review Guidelines 3.1.2(b)에서 구독 서비스는 해지 방법을 명확히 제공해야 한다고 규정한다. 이를 준수하지 않으면 앱 심사 거절 사유가 된다.
> - **신뢰 구축**: 해지 방법을 숨기지 않고 투명하게 안내하는 것은 오히려 사용자 신뢰를 높인다. 역설적으로, 해지가 쉽다는 확신이 가입 전환율을 높이는 효과가 있다.
> - **데이터 보존 안내**: "기록과 뱃지는 유지된다"는 정보는 해지 고민 중인 사용자에게 "나중에 다시 가입하면 처음부터 시작하지 않아도 된다"는 안심을 주어, 재가입 가능성을 열어둔다.

## 3. UX 설계 의도

### 3.1 정보 계층 구조

```
Level 1 (감정): 히어로 섹션 — 감정적 설득 ("특별한 경험")
Level 2 (비교): 비교 표 — 논리적 설득 ("이것이 차이")
Level 3 (상세): 혜택 카드 — 구체적 가치 ("이렇게 좋다")
Level 4 (행동): 가격 + CTA — 행동 유도 ("지금 시작")
Level 5 (해소): FAQ — 불안 해소 ("걱정 마세요")
Level 6 (안전): 해지 안내 — 신뢰 구축 ("언제든 그만둘 수 있어요")
```

이 구조는 AIDA 모델(Attention-Interest-Desire-Action)을 확장한 설득 흐름이다. 히어로(Attention) → 비교표(Interest) → 혜택 카드(Desire) → 가격/CTA(Action) → FAQ/해지(Trust)의 순서로, 사용자를 단계적으로 결제 행동까지 이끈다.

### 3.2 인터랙션 패턴

| 인터랙션 | 동작 | 피드백 |
|---------|------|--------|
| 플랜 카드 선택 | 카드 탭 | 선택 카드 Primary 테두리 + 체크마크, 비선택 카드 Gray 테두리 |
| CTA 버튼 탭 | 버튼 터치 | 버튼 스케일 애니메이션 + OS 결제 시트 호출 |
| FAQ 아코디언 | 질문 탭 | 답변 영역 슬라이드 다운/업 (300ms ease) |
| 스크롤 | 상하 스와이프 | CTA 버튼이 화면 밖으로 나가면 하단 고정 CTA 바 표시 |
| 닫기 | 좌상단 X 또는 뒤로가기 | 페이지 닫힘, "나중에 결정하시겠어요?" 확인 (선택적) |

### 3.3 접근성 고려사항

- **VoiceOver/TalkBack**: 가격 정보, CTA 버튼에 명확한 `accessibilityLabel` ("월 3,900원 프리미엄 구독 시작 버튼")
- **비교 표 접근성**: 표를 접근성 트리에 적절히 표현 (헤더/데이터 셀 구분)
- **시각 장애**: 할인 리본의 "33% 할인" 정보를 텍스트로도 명시
- **터치 영역**: CTA 버튼 최소 56px 높이, 플랜 카드 전체가 터치 영역
- **다크 모드**: 그라데이션, 카드 배경, 텍스트 색상 다크 테마 전환
